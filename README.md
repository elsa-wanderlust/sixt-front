## SET UP 👨‍💻

```bash
npm install
```

or

```bash
npm i
```

to install all the packages

```bash
npm run dev
```

to start the project

open `http://localhost:5173/` in your browser, and start booking cars!

The backend repo is available [here](https://github.com/elsa-wanderlust/sixt-backend)
